cat foothold

At this point I got stuck for a while because I couldn't figure out the next step, so I went back to carefully examining the original first portal.

Reading more carefully, I had missed a relevant piece of information on the homepage:

They are using fonttools. I search online for known issues and come across CVE-2025-66034. This CVE allows creating arbitrary PHP files inside the webroot directory of the fonttools installation. The issue is that I don't know what that folder is, but by exploiting the Path Traversal vulnerability I can dump the application source code to investigate further.

The file to download is clearly `download.php` itself since, at least in theory, it should be downloading fonts generated from that very directory:

require_once 'auth.php';
require_login();

$file = $_GET['f'] ?? '';
if (!$file) {
    die('File parameter required.');

$file = str_replace("../", "", $file);

Among other things, I notice that /opt contains a file with a .bak extension, which is typically used for backup copies.

I dump the contents of this file and discover it is a script used to manage user uploads on the main site. This kind of file is usually triggered by either a cronjob or a watch it's not clear yet. Since I don't have permission to read /etc/crontab, I push pspy to the target machine, launch it to observe process activity, reload the previously used files, and wait.

After a few seconds I see a file process_client_submissions.sh (which should be the original copy of the .bak) being executed inside steve's home directory.

Considering that only steve can use their own home directory:

The install_validator_plugin method downloads a plugin from a URL and places it in a PLUGIN_DIR folder via the index.download method of PackageIndex class. A Google search reveals that this class has a Path Traversal vulnerability described in CVE-2025-47273. The previously cloned repository already includes a PoC for this CVE, but I chose to do it manually by overwriting the /etc/sudoers file instead of adding a known key:
